# tokio-rmp-codec

## Installation

To use, add the following line to `Cargo.toml` under `[dependencies]`:

```toml
tokio-rmp-codec = { git = "https://github.com/enygmator/tokio-rmp-codec.git" }
```

## Usage

Refer the unit and integration tests for usage.
